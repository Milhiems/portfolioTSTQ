Gazette interactive

Note explicative

La gazette interactive a été réalisée dans le but d’être liée à la plateforme d’écriture sur laquelle j’ai travaillé. La volonté était de donner des informations de lore aux joueur·euse·s concernant des événéments récents dans l'intrigue de l'univers. Le but était de rappeler un écran d'ordinateur avec différentes fenêtres ouvertes. Ce format a été choisi dans le but de permettre aux joueur·euse·s de fouiller dans les éléments, et s'inscrivait davantage dans le lore de l'univers mis en place qu'un autre format plus classique.
La conception de ce projet a été faite assez rapidement, environ 48h pour les maquettes & le code.

DESIGN & DÉVELOPPEMENT FRONT-END

J’étais chargé de la construction de la structure, basée sur une maquette réalisée sur Figma par Camille MISBACH. J’ai utilisé pour cette gazette les langages HTML, CSS et JavaScript.

RETOURS

Ce projet étant été réalisé très rapidement, j'aurais peut-être aimé pouvoir modifier certaines choses pour une seconde version : la possibilité de ranger les éléments avec une barre des tâches pour rester dans l'idée d'un écran d'ordinateur, peut-être également de reproduire plus en détail l'interface d'un ordinateur avec les différentes animations, les menus, etc.
